# Sparrowhawk

A Laravel sub-framework and UI plugin ecosystem.

> *"From apprentice to mage of the web."*

## Status

**Preview** — design in progress. This repository is the foundation for the Sparrowhawk framework and its ecosystem. APIs and structure are not yet stable.

## Vision

Sparrowhawk extends Laravel with batteries-included conventions for building polished web applications, paired with a curated catalog of UI plugins that share a coherent design language.

The ecosystem includes:

- **`sparrowhawk-labs/sparrowhawk`** — the framework core (this repository)
- **`sparrowhawk-labs/laravel-sparrowhawk`** — Laravel integration
- **`sparrowhawk-labs/sparrowhawk-ui`** — UI plugin collection
- **`sparrowhawk-labs/sparrowhawk-studio`** — visual builder and admin tooling

## Inspiration

The project draws its name from Ursula K. Le Guin's *Earthsea* — a story about a wizard whose journey from apprentice to master mirrors the path of a developer learning to wield powerful tools with care.
